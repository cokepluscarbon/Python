#1. 文件和IO

文本和IO操作是编程语言普通不过的功能，Python提供了很多便捷的方法对文件进行IO操作。

##1.1 读写文本数据

> 使用open()方法的rt模式读取文本文件

```python
with open('somefile.txt', 'rt') as f:
    data = f.read()
    print (data)
```
read()读取整个文件并作为字符串返回，我们也可以一行行地读取文本文件，使用<code> for 变量名 in 可迭代对象</code>即可：

```
with open('C://log_client.txt', 'rt') as f:
    for line in f:
        print (line)
```
